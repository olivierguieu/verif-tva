About

This service is provided thanks to the online service provided by EC DG TAXUD. First VAT numbers are verified against a regular expression and can then be verified through a call to the VIES web service.

The following tools, services were instrumental building VerifTVA:

  • SUDZC for building the objective C SOAP client code
  • inAppSettingsKit for managing the settings in and outside the application
  • Apple "Reachability" sample code
  • Some code to setup defaults for user settings
  • CountryPicker for iOS

For any additional information send me an e-mail.

A propos

Ce service est fourni grace au web service maintenu par le service de Fiscalité et Union douanière de la Commission Européenne . Les numéros de TVA saisis sont d'abord validés via une expression régulière puis peuvent être validés en appelant le web service "VIES".

Les outils suivants ont été utilisés pour développer l'application:

  • SUDZC pour générer le code du client du service SOAP en objective C
  • inAppSettingsKit pour gérer le paramétrage utilisateur
  • Apple "Reachability"
  • Initialisation du paramétrage par défaut
  • CountryPicker pour IOS

Pour toute question supplémentaire envoyez moi un e-mail.

A proposito

Questo servizio è fornito grazie al servizio online fornito da CE DG Fiscalità e unione doganale . Numeri di partita IVA primi sono verificati a fronte di una espressione regolare e possono quindi essere verificata attraverso una chiamata al servizio Web VIES.

I seguenti strumenti, servizi sono usati in VerifTVA :

  • SUDZC per generare il client del servizio SOAP in Objective C
  • inAppSettingsKit per gestire le impostazioni utente
  • Apple "Reachability"
  • Inizializzare l'impostazione predefinita
  • CountryPicker per iOS
